# e-SUS AF

Protótipo montado para o e-SUS AF com PPU UNASUS - FIOCRUZ.

## Sobre

Este projeto é um protótipo desenvolvido para o sistema e-SUS AF (Atenção Primária) em parceria com PPU UNASUS (Universidade Aberta do SUS).

## Objetivo

Desenvolver e testar funcionalidades relacionadas ao sistema e-SUS AF, uma ferramenta do Ministério da Saúde para gestão da Atenção Primária à Saúde.

## Como Usar

### Pré-requisitos
- Verificar dependências específicas do projeto
- Acesso ao ambiente e-SUS AF (se necessário)

### Instalação

```bash
# Clonar o repositório
git clone https://github.com/ThiagoMarques/esusaf.git

# Instalar dependências
npm install
# ou conforme especificado no projeto
```

### Execução

```bash
# Iniciar o servidor/ aplicação
npm start
```

## Sobre o e-SUS AF

O e-SUS Atenção Primária (e-SUS AB) é um sistema desenvolvido pelo Ministério da Saúde para informatizar a Atenção Primária à Saúde, permitindo:

- Cadastro de cidadãos
- Registro de atendimentos
- Gestão de equipes de saúde
- Produção de relatórios e indicadores
